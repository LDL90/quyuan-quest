# 🐉 端午屈原主题 · 公园探秘寻游

> 亲子互动项目 — 扫码闯关 · 边玩边学 · 端午限定

---

## 一、项目概述

| 项目 | 说明 |
|------|------|
| 主题 | 「寻找屈原的足迹」— 端午文化探秘 |
| 形式 | 公园实地闯关 + 手机扫码互动 |
| 时长 | 约 60-90 分钟 |
| 适合年龄 | 5-12 岁亲子家庭 |
| 成本 | ≈ 0 元（仅需打印二维码贴纸） |

### 核心玩法
1. 家长带孩子到公园，扫码进入互动页面
2. 每个关卡对应公园里的一个标志性物品/地点
3. 孩子完成任务（答题/拍照/找物），解锁下一关
4. 全部通关获得「屈原文化小达人」电子证书

---

## 二、屈原主题关卡设计（6关）

### 🏠 关卡1：汨罗江畔 — 公园入口/湖边

**故事引导：**
> 屈原站在汨罗江边，望着远方，心中满是忧愁。小朋友，你知道屈原是哪个国家的人吗？

**公园对应点：** 公园入口牌坊 / 人工湖边

**任务：** 选择题 — 屈原是哪个国家的人？
- A. 秦国 ❌
- B. 楚国 ✅
- C. 齐国 ❌
- D. 燕国 ❌

**通关提示：** 「屈原是楚国的三闾大夫，他深爱着自己的国家！」

**互动加分项：** 在湖边拍一张"望江照"，模仿屈原远眺的样子

---

### 🌿 关卡2：艾草飘香 — 花坛/草坪区

**故事引导：**
> 端午节到了，家家户户门前都挂上了艾草。传说艾草可以驱邪避害，保护家人平安。小朋友，你能找到公园里的一种香草或特别的植物吗？

**公园对应点：** 花坛 / 植物标识牌旁

**任务：** 拍照任务 — 拍下公园里你最喜欢的一种植物

**知识卡：**
> 端午挂艾草的习俗已有千年。古人认为五月是"毒月"，艾草的香气可以驱赶蚊虫和邪气。屈原在《离骚》中写道："扈江离与辟芷兮，纫秋兰以为佩"——他也很喜欢香草呢！

---

### 🐉 关卡3：龙舟竞渡 — 长廊/步道

**故事引导：**
> 听！远处传来鼓声——咚！咚！咚！那是龙舟赛的鼓点！传说屈原投江后，人们划龙舟去救他，后来就变成了端午赛龙舟的习俗。小朋友，你知道龙舟上最少需要几个人吗？

**公园对应点：** 长廊 / 曲折步道（模拟龙舟赛道）

**任务：** 选择题 — 龙舟上最少需要几个人？
- A. 2人（一个划船一个打鼓）✅
- B. 10人 ❌
- C. 20人 ❌

**体能互动：** 从长廊这头快步走到那头，模拟划龙舟！计时挑战

**通关提示：** 「龙舟竞渡，百舸争流！团结协作才能最快到达！」

---

### 📜 关卡4：离骚长歌 — 亭子/休息区

**故事引导：**
> 屈原写下了千古名篇《离骚》，这是中国最长的抒情诗。他写道：「路漫漫其修远兮，吾将上下而求索」——意思是，路还很长，但我要坚持探索！小朋友，我们来接句诗吧！

**公园对应点：** 亭子 / 休息长椅区

**任务：** 填空题 — 「路漫漫其修远兮，____而求索」
- 答案：上下

**趣味挑战：** 家长和孩子一起大声朗读这句诗3遍！感受屈原的精神

**知识卡：**
> 《离骚》共373句、2490字，屈原用香草美人比喻高洁的品格。他创立了"楚辞"这一诗歌体裁，被称为"辞赋之祖"。

---

### 🥟 关卡5：粽香传情 — 石桌/野餐区

**故事引导：**
> 传说屈原投江后，百姓怕鱼虾吃他的身体，就把米团投入江中，后来演变成了包粽子。粽子有甜有咸，你喜欢吃哪种呢？不管甜咸，包的都是对屈原的纪念！

**公园对应点：** 石桌石凳 / 野餐草坪

**任务：** 选择题 — 最早的粽子是什么口味？
- A. 甜粽 ❌
- B. 咸粽 ❌
- C. 没有馅的纯米粽 ✅

**互动加分项：** 拍一张家庭合影，比出最开心的表情！

**知识卡：**
> 最早的粽子叫"角黍"，是用菰叶包黍米做成牛角状。现在的粽子五花八门：肉粽、豆沙粽、蛋黄粽、碱水粽……你家乡包什么粽？

## 三、技术实现方案

### 方案A：纯H5网页（推荐新手，零成本）

```
技术栈：纯 HTML + CSS + JavaScript（无需服务器）
部署：GitHub Pages（免费）或 任何静态托管
二维码：每个关卡一个URL → 生成对应二维码
```

**文件结构：**
```
quyuan-quest/
├── index.html          # 首页/开始页
├── quest/
│   ├── level1.html     # 汨罗江畔
│   ├── level2.html     # 艾草飘香
│   ├── level3.html     # 龙舟竞渡
│   ├── level4.html     # 离骚长歌
│   ├── level5.html     # 粽香传情
│   └── level6.html     # 屈原精神
├── certificate.html    # 电子证书页
├── css/
│   └── style.css       # 全局样式
├── js/
│   └── app.js          # 互动逻辑
└── assets/
    └── images/         # 装饰图片
```

**二维码生成方式：**
- 每个关卡页面对应一个URL
- 用在线工具（如草料二维码）生成二维码
- 打印成贴纸/卡片，放在公园对应位置

### 方案B：微信小程序（体验更好，需开发）

```
技术栈：微信小程序原生开发 / uni-app
优势：可分享、可保存证书、用户体验更流畅
成本：微信小程序认证 ¥300/年（个人免费）
```

### 方案对比

| | H5网页 | 微信小程序 |
|---|---|---|
| 开发难度 | ⭐⭐ 简单 | ⭐⭐⭐⭐ 较难 |
| 成本 | 0元 | 0-300元/年 |
| 用户体验 | 良好 | 优秀 |
| 分享传播 | 链接分享 | 小程序卡片 |
| 离线使用 | 需联网 | 部分支持 |
| 上架时间 | 即时 | 需审核1-7天 |
| **推荐** | ✅ 首选起步 | 后期升级 |
